# 🔋 EV Charging Dashboard - Electric Vehicle Management

A comprehensive dashboard for managing and monitoring electric vehicle charging stations. Track charge levels, find available stations, and optimize charging schedules.

## ✨ Key Features

- 🗺️ **Station Locator** - Find nearby EV charging stations
- 📊 **Real-time Monitoring** - Track charging progress
- ⚡ **Battery Status** - Monitor vehicle battery health
- 💰 **Cost Tracking** - Calculate charging expenses
- 🌍 **Station Database** - Browse all available stations
- ⏰ **Scheduling** - Plan charging sessions
- 📈 **Statistics** - Track charging history
- 📱 **Responsive Design** - Use on any device

## 🛠️ Tech Stack

**Frontend:**
- HTML5
- CSS3 (responsive design)
- JavaScript
- Interactive maps

**Features:**
- Location services
- Real-time data updates
- Data visualization
- Responsive layout

## 🚀 Getting Started

### Prerequisites
- Modern web browser with Geolocation API support
- No installation required

### Installation

```bash
# Clone the repository
git clone https://github.com/deyjaydeep40-arch/EVchargingdashboard.git
cd EVchargingdashboard

# Open in browser
open index.html

# Or use a local server
python -m http.server 8000
# Visit http://localhost:8000
```
